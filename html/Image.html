Image
Start a Image object.

KEYS:
type
What type of image/icon set to use.
[string] |toml string| : https://toml.io/en/v1.0.0-rc.1#string

src
path to image if image, material icon name if material icon
[string] |toml string| : https://toml.io/en/v1.0.0-rc.1#string

height
CSS height of the image
[size] |css size| : https://developer.mozilla.org/en-US/docs/Web/CSS/@page/size


|Shorthand available|
Simply provide the following:
-[string] path to image or material icon name-
and it will use the defaults for the rest of the keys.
{type = '${1|image,material-icon|}', src = '${2|/static/images/apps/default.png,/static/user_assets/images/default.png,material_icon_name,https://,http://|}', height = '${3|64px,2rem,10%|}'}
