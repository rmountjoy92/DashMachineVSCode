Onpress
Start a Onpress object.

KEYS:
href
The url or dashboard to navigate to
[string] |toml string| : https://toml.io/en/v1.0.0-rc.1#string

target
Open the url in this tab, a new tab, or iframe, or navigate to a dashboard
[string] |toml string| : https://toml.io/en/v1.0.0-rc.1#string


|Shorthand available|
Simply provide the following:
-[string] Url to navigate to-
and it will use the defaults for the rest of the keys.
